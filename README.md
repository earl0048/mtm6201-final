# mtm6201-final
Final Project for 26W_MTM6201_020 Advanced Web Design with HTML and CSS class at Algonquin College, in the Interactive Media Design program, in level 2.

This project is by Ashlee Earle.

This project is attempting to replicate my figma made web design. 

This project was based on an assignment in another class where I had to design a website with 3 pages, that was responsive in Figma. The website is for a fake company that runs an activity/community centre. I was provided with various materials as if the comany had done some interviews with memembers of the community centre such as customers staff, vendors and other people who interect with the centre. I then had to create a some personas, reserach websites, ask what would be necessairy out of 3 pages for a brand new website/company, what would make it appealing etc. I then created a mood board. Then a style tile. Then a site map. Then wireframes. Then a low-fidelity mock-up. Then high-fidelity mock up. Then finally an interactive prototype. My Figma prototype is not great for accessibility so I will address that while still maining the brand's look and feel and using the same colours but in a more accesible way.
